# <a href="https://thefashionstore.netlify.app/">jfs-ecommerce</a>
An ecommerce website.
Built with:
- Persistence: MySQL Database.
- Backend: REST API with Java and Spring Boot.
- Frontend: JavaScript + CSS + Bootstrap + HTML
